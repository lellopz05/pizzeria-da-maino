# Da Maino — sito web

Sito one-page per Pizzeria Ristorante Enoteca da Maino, Conco (VI).

## Asset necessari

Metti in `assets/` queste immagini:

| File | Descrizione |
|------|-------------|
| `hero-bg.jpg` | Foto esterno locale (cover FB) — 1920×1080+ |
| `logo.png` | Logo cerchio giallo/nero — 500×500+ |
| `interno.jpg` | Sala interna (mattoni, legno, camino) |
| `enoteca.jpg` | Bar/enoteca con bottiglie |
| `dehors.jpg` | Dehors coperto con tovaglie |
| `birreria.jpg` | Birreria esterna su prato sintetico |
| `piatto-altopiano.jpg` | Piatto Unico Altopiano |
| `pizza-toro.jpg` | Pizza Toro Del Maino |
| `pizza-classica.jpg` | Pizza classica |

> Le foto FB non sono scaricabili via script (privacy block). Serve richiederle al titolare o rifare scatti.

## Deploy

Statico puro — apri `index.html` in browser o hosta su qualsiasi static host (Netlify, Vercel, GitHub Pages).
